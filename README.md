# Task Tracker CLI

Project URL https://roadmap.sh/projects/task-tracker

## How to run

Clone the repository and run the following command:

```
git clone https://github.com/TEKnas29/Task_tracker.git
cd Task_tracker
```
Requirements
```
Bun
npm
node
```
Run the following command to run the project:

```
npm i
npm run exec
```
